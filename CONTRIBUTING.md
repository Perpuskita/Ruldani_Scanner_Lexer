# Introduction 
Hello !! Thank you for contributing.

# Tata cara kontribusi
tambahkan issue, fork, pull request dan tunggu review dari owner
Sambil baca bisa ngisi folmulir kontribusi disini :

issue            : #_______ # isi dengan taggar dari issue yang sudah ditambahkan
tipe kontribusi  : feat, fix, docs, chore # pilih salah satu
deskripsi        : # deskripsikan perubahan apa yang telah dilakukan 



# Catatan untuk kontributor

fix   : untuk memperbaiki error atau bug
feat  : menambahkan fitur baru ( termasuk juga melakukan refactor )
docs  : menambahkan dokumentasi terkait project termasuk readme.md
chore : untuk mengganti nama variable dan 

owner bebas contrib ngga pake issue loh ya wkkw
